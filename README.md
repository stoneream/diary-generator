# diary-generator

## install

```bash
go install github.com/stoneream/diary-generator/v2@latest
```

## usage

```bash
# ベースディレクトリの作成
mkdir diary

# テンプレートファイルの作成
touch template.md

# 初期化

diary-generator init

# アーカイブ

diary-generator archive --target-ym 2023-01

# サマリ

diary-generator summary

cd archive
diary-generator summary --target-prefix diary_
```
