# diary-generator

## usage

```bash
# init

java -jar bin/diary-generator.jar init --base-directory ./diary --template-path bin/template/diary.md 

# archive

java -jar bin/diary-generator.jar archive --base-directory ./diary --starts-with 2023-01
```
